Весь российский рынок копицентров сидит в синем — **он занят и серый**.
FedEx Office доказал: `глубокий фиолетово-синий + оранжевый/янтарный` = доверие × внимание.
В России эта комбинация **свободна**. Мы берём её.

## 2. Психология цвета

### Почему Deep Indigo (#1E1B4B / #312E81)
- Наследует ДОВЕРИЕ от синего (70% людей называют синий «надёжным»)
- Добавляет ПРЕМИУМ от фиолетового (ассоциация: качество, глубина)
- «Чернила» — прямая метафора печати
- Выделяется на фоне конкурентов-«синяков»

### Почему Amber #D97706 (янтарный/оранжевый)
- Кнопки оранжевого/янтарного цвета конвертируют на 32,5% лучше синих (A/B тесты HubSpot)
- Создаёт «тепло» — ощущение только что напечатанного, тёплого листа
- Срочность БЕЗ агрессии (в отличие от красного)
- Комплементарен к индиго → максимальный визуальный контраст = внимание захвачено
- Редкость в нише = запоминаемость

### Синергия
```
Индиго (тёмный) + Янтарный (яркий) = "Быстрая, премиальная печать которой можно доверять"
```

## 6. Правила применения

### Кнопки
```
 PRIMARY (главное действие):
   bg: amber-600   hover: amber-700   text: white
   Пример: "Новая заявка", "Создать", "Сохранить", "Войти"

 SECONDARY (второстепенное):
   bg: gray-100   hover: gray-200   text: gray-700
   Пример: "Отмена", "Назад"

 DANGER (удаление):
   bg: red-50   hover: red-100   text: red-600
   Пример: "Удалить"
```

### Шапка
```
 Фон: Brand Ink #1E1B4B (indigo-950)
 Текст: белый
 Акцент иконки бренда: янтарный
 Навигация: indigo-300 → белый при hover
```

### Активные состояния
```
 Фильтры-чипсы: amber-600 текст белый
 Текущий статус: amber-600 текст белый
 Активная категория: border amber-500, bg amber-50
```

### Форма ввода
```
 Border по умолчанию: gray-200
 Focus ring: ring-amber-500 (янтарный ring = привлекает внимание)
 Label: gray-700
```
