# Портфолио работ

Сюда складывайте реальные фотографии готовых футболок.

## Как подключить фото

1. Положите файл, например `work1.jpg`, в эту папку.
2. В `src/landing/data/portfolio.ts` импортируйте его и укажите в поле `src`:

```ts
import work1 from '../../assets/portfolio/work1.jpg';

export const portfolioItems = [
  { id: '1', src: work1, caption: 'Подарок, который не ожидали', alt: 'Девушка с футболкой с фото — печать в …', color: '#fff', ink: '#1E1B4B' },
  // …
];
```

Если `src` не указан — рисуется SVG-заглушка футболки. Остальной код
(галерея, lazy loading, lightbox, alt) менять не нужно.

## Рекомендации
- Формат: JPG/WebP, квадрат ~800×800, вес < 200 КБ.
- Обязательно заполняйте `alt` с локальным ключом (город) для SEO.
